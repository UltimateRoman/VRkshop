Mechanical Engineering Workshop

Experiment 2: Fitting


Aim: To make a step fitting.



Tools Required:


Ball-peen hammer


Hack-saw


Rasp file



Procedure:

1. Using the vernier height gauge and the v-block with clamp, mark the required dimensions on the rectangular block.
2. Mark dots on the boundary lines of the pieces to be cut out from the block using a dot punch and a ball peen hammer.
3. Plane 2 adjacent surfaces of the wooden block using a metal jack plane.
4. Mark the required dimensions on the block with respect to planed surfaces using steel rule and pencil.
5. Plane the remaining surfaces using the metal jack plane into required dimensions.


Enter Lab


Result: The step fitting of required dimensions is made.
