Mechanical Engineering Workshop

Experiment 1: Carpentry


Aim: To cut and plane a wooden block into required dimensions.



Tools Required:

Metal Jack Plane


Chisel


Rasp file



Procedure:

1. Fix the wooden block firmly on the carpenter's vice.
2. Cut it into the required length using steel rule and pencil.
3. Fix the block on bench vice and using the hacksaw, divide it into 2 pieces through the middle.
4. Using hacksaw, carefully cut away excess portions from the two pieces to be obtained.
5. Using the file, file away the excess portions and smoothen the edges till required dimensions are obtained.


Enter Lab


Result: A plane wooden block of required dimensions is obtained.
